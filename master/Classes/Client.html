Client

open class Client

Client instances are the gateway to access services.

Usually you wouldn’t instantiate the Client class directly, but one of the classes inheriting from Client. Such inheriting classes are generated by Lark from WSDL provided by the service. These generated classes contain typed operations as defined in the WSDL.

However advised against, you could also use a Client instance directly to pass messages to a service. Be aware that the API of Client might change without further warning.
